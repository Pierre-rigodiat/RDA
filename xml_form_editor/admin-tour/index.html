Schema View Configuration

This feature allows system administrators to fully configure the form. This includes:

  • The total number of pages,
  • The page on which individual elements will be displayed,
  • The plugin modules which are loaded with the form,
  • The assignment of form elements to be rendered by each module.
After configuration, the form can be saved to the system database for use with the form.
